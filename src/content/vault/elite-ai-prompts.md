## MODULE 1: WRITING PROMPTS

### Content Writing

**Article Scaffolding:**
```
Write a detailed outline for a [WORD_COUNT]-word article titled "[TITLE]" targeting [AUDIENCE]. 
Include H2s, H3s, key statistics placeholders, and a compelling intro hook.
```

**Long-Form Blog Post:**
```
Write an authoritative [WORD_COUNT]-word blog post on "[TOPIC]". 
Target keyword: [KEYWORD]. Tone: [professional/conversational/technical]. 
Include: strong hook, 3–5 H2 sections, actionable takeaways, and a CTA.
```

**Headline Generation:**
```
Generate 20 high-CTR headlines for an article about "[TOPIC]". 
Use these frameworks: How-To, Listicle, Question, Curiosity Gap, Number-Led. 
Target audience: [AUDIENCE].
```

**Story-Led Content:**
```
Write a compelling story that illustrates the transformation from [BEFORE_STATE] 
to [AFTER_STATE] for [AUDIENCE]. Use the AIDA framework. Length: [WORD_COUNT] words.
```

**Expert Roundup Article:**
```
Simulate 5 expert opinions on "[TOPIC]" from the perspectives of: 
[EXPERT_1], [EXPERT_2], [EXPERT_3], [EXPERT_4], [EXPERT_5]. 
Format as quotes with attribution and 2-sentence context each.
```

### Copywriting

**Facebook/Instagram Ad Copy:**
```
Write 5 Facebook ad variations for [PRODUCT/SERVICE]. 
Target: [AUDIENCE]. Benefit: [PRIMARY_BENEFIT]. Pain: [CORE_PAIN].
Format: Hook (1 line) | Body (3 lines) | CTA. Include emoji variants.
```

**Google Ad Headlines:**
```
Generate 15 Google Search Ad headlines (max 30 characters each) for [PRODUCT].
Target keyword: [KEYWORD]. USP: [UNIQUE_SELLING_PROPOSITION].
```

**Sales Page Copy:**
```
Write a full sales page for [PRODUCT/OFFER]. Include:
- Power headline + subheadline
- Problem agitation section
- Solution introduction
- 5 key benefits (outcome-focused)
- Social proof placeholders
- FAQ section (5 questions)
- CTA repeated 3x
Tone: [TONE]. Price: [PRICE].
```

**Product Description:**
```
Write 3 product description variants for [PRODUCT] targeting [AUDIENCE].
Variant 1: Feature-led (technical). Variant 2: Benefit-led (emotional). 
Variant 3: Story-led (narrative). Max 150 words each.
```

**Cold Email Sequence (5-part):**
```
Write a 5-email cold outreach sequence for [SERVICE] targeting [ICP].
Pain point: [PAIN]. Value prop: [VALUE]. 
Email 1: Intro/hook. Email 2: Case study. Email 3: Objection handling. 
Email 4: Urgency. Email 5: Break-up. Subject lines included.
```

### Creative Writing

**Short Story Generator:**
```
Write a [GENRE] short story (800 words) with:
- Protagonist: [CHARACTER_TRAITS]
- Setting: [SETTING]
- Central conflict: [CONFLICT]
- Resolution type: [OPEN/CLOSED/TWIST]
Opening line must create immediate tension.
```

**Character Development:**
```
Create a detailed character profile for [CHARACTER_NAME] in a [GENRE] story.
Include: backstory, core motivation, fatal flaw, speech pattern, 
relationship dynamics, and 3 defining moments.
```

**Dialogue Writing:**
```
Write a tense 20-line dialogue between [CHARACTER_1] and [CHARACTER_2] 
about [TOPIC/CONFLICT]. Each character has a distinct voice: 
[CHARACTER_1_VOICE] vs [CHARACTER_2_VOICE]. No action beats, pure dialogue.
```

## MODULE 2: MARKETING PROMPTS

### Strategy & Research

**ICP Research:**
```
Build a detailed Ideal Customer Profile for [BUSINESS/PRODUCT].
Include: demographics, psychographics, daily frustrations, goals, 
channels they use, content they consume, and objections to buying.
Output as a structured profile card.
```

**Competitor Analysis:**
```
Analyze [COMPETITOR] using this framework:
- Positioning statement
- Top 5 messaging themes
- Identified weaknesses
- Opportunities to differentiate
- 3 angles [MY_BRAND] can exploit
Base analysis on publicly available information for [COMPETITOR].
```

**Content Strategy:**
```
Build a 90-day content strategy for [BRAND] in [NICHE].
Goal: [AWARENESS/LEADS/SALES]. Channels: [CHANNELS].
Output: Monthly themes → Weekly content pillars → 3 post ideas per pillar.
Include content formats for each channel.
```

**SEO Keyword Clusters:**
```
Generate keyword clusters for [TOPIC] targeting [AUDIENCE].
Group: Primary keywords (≥1000 monthly searches) | Secondary keywords | 
Long-tail (buyer intent). Format as a table with search intent tags.
```

**Market Entry Strategy:**
```
Create a market entry strategy for launching [PRODUCT] in [MARKET].
Cover: Target segment, positioning, pricing model, go-to-market channels,
first 30-day launch plan, and 3 key success metrics.
```

### Social Media

**LinkedIn Personal Brand Post:**
```
Write a LinkedIn post using the "Insight → Story → Takeaway" framework.
Topic: [TOPIC]. Target: [AUDIENCE]. Tone: [AUTHORITATIVE/VULNERABLE/TACTICAL].
End with a question to drive comments. Max 1300 characters. No hashtags.
```

**Twitter/X Thread:**
```
Write a 15-tweet thread on [TOPIC] for [AUDIENCE].
Tweet 1: Bold claim/hook. Tweets 2–13: One insight per tweet with proof.
Tweet 14: Summary. Tweet 15: CTA + self-referential tweet.
Format for max engagement on X.
```

**Instagram Caption Pack (7 days):**
```
Write 7 Instagram captions for [BRAND/NICHE] covering:
Day 1: Educational. Day 2: Behind-the-scenes. Day 3: Testimonial format. 
Day 4: Question/poll. Day 5: Motivational. Day 6: Product/service.
Day 7: Community appreciation. Include emoji and CTA for each.
```

**YouTube Script (10 min):**
```
Write a 10-minute YouTube script on "[TITLE]" for [AUDIENCE].
Structure: Hook (30s) → Problem (1min) → Credibility (30s) → 
3 main points (2min each) → Recap (30s) → CTA (30s).
Include B-roll suggestions in brackets. Conversational tone.
```

**TikTok Hook Library (10 hooks):**
```
Generate 10 TikTok video hooks for content about [TOPIC].
Each hook must: Start with a pattern interrupt, be max 2 sentences,
stop the scroll within 2 seconds. Formats: question, statement, challenge, reveal.
```

## MODULE 3: SALES PROMPTS

### Objection Handling

**Price Objection:**
```
Write 5 responses to the objection "It's too expensive" for [PRODUCT] priced at [PRICE].
Each response should use a different technique: reframe, ROI calculation, 
comparison, payment plan, and value stack. Keep each under 50 words.
```

**"I need to think about it":**
```
Write 3 sales responses to "I need to think about it" for [PRODUCT/SERVICE].
Response 1: Uncover real objection. Response 2: Future pacing. 
Response 3: Risk reversal + urgency. Sales context: [CONTEXT].
```

**"Not the right time":**
```
Write a script for handling "the timing isn't right" for [OFFER].
Include: empathy statement, future cost of inaction calculation,
bridge to decision, and soft close. Max 100 words.
```

### Closing Scripts

**Discovery Call Script:**
```
Write a 30-minute discovery call script for selling [SERVICE] to [ICP].
Sections: Rapport (3min) | Situation questions (8min) | Pain exploration (8min) |
Vision casting (5min) | Transition to pitch (3min) | Next steps (3min).
Include exact question language.
```

**DM Sales Script:**
```
Write a 5-message DM sales sequence for selling [OFFER] on [PLATFORM].
Message 1: Warm opener. Message 2: Value delivery. Message 3: Discovery question.
Message 4: Soft pitch. Message 5: Follow-up + close. No spam tactics.
```

**Proposal Template:**
```
Write a proposal template for [SERVICE] sent to [CLIENT_TYPE].
Sections: Executive summary | Problem acknowledged | Proposed solution |
Deliverables & timeline | Investment options (3 tiers) | Next steps.
Professional tone. Outcome-focused language.
```

### Customer Success

**Testimonial Request:**
```
Write 3 versions of a testimonial request for [PRODUCT/SERVICE].
Version 1: Email. Version 2: Text/SMS. Version 3: Post-call verbal script.
Guide customers to answer: What was the problem? What changed? 
What result did you get? Would you recommend?
```

**Upsell Script:**
```
Write an upsell script for offering [UPSELL_PRODUCT] to customers of [CORE_PRODUCT].
Trigger: [POST_PURCHASE/MILESTONE/TIME_BASED]. 
Frame as a natural next step. Include the transition phrase.
```
