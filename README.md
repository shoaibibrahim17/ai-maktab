# SI | AL-HIKMA: The Ibrahim Maktaba

This is a premium, physics-based AI resource hub built with React and Framer Motion. 
Designed with an elegant 'Islamic-Minimalist' Deep Emerald and Gold aesthetic.

## 🚀 Vision
To provide a cinematic, immersive interface for exploring high-value AI resources under the banner of Al-Hikma (The Wisdom).

## 🛠 Tech Stack
- **React**: Component-based UI.
- **React Router**: Client-side navigation to the markdown vault viewer.
- **Framer Motion**: Realistic spring-based physics for 3D card tilt interactions.
- **Vanilla CSS**: Custom glassmorphism, responsive CSS grid, and atmospheric depth.
- **Vite**: Ultra-fast development server.

## ✨ Design Principles
1. **The Treasury (Khizana/Maktaba)**: Layered translucency, Emerald glow, and Gold calligraphy styling.
2. **Weightless Interaction**: Hovering cards float upwards using spring physics.
3. **Advanced Vault Viewer**: Embedded local markdown parsing and dynamic routing.

## 📁 Project Structure
- `src/components`: UI components like Header, ResourceCard, and VaultViewer.
- `src/content/vault`: The local Markdown resource library.
- `src/data`: Contains `resourceData.json` with the underlying data routing structure.
- `src/index.css`: The core design system and vault markdown CSS rules.
